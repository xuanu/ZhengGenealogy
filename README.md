# ZhengGenealogy  
主要记录贵州省铜仁市德江县郑氏族谱      
> emmx格式的文件可使用MindMaster打开，可编辑，可导出。       
> 记录软件为MindMaster,可导出图片，PDF等等格式。         
     
可在总谱在找到下列各个地方的族谱：            
七里溪：郑启->郑顺国->郑登弟->郑之?1（不详）      
苦竹园：郑启->郑顺国->郑登弟->郑之泰        
梨树坪（一）：郑启->郑顺国->郑登昌->郑大禹->郑德时->郑文武     
梨树坪（二）：郑启->郑顺国->郑登昌->郑大轮（梨树坪二未记录始祖线，故记入郑登昌下）        
坝坪岩角角： 郑启->郑顺国->郑登弟->郑之道->郑大章->郑德印->(1文武，2文堂)
